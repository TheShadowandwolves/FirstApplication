What is this?

Hi to everyone who is entering this website.
This website is a personal project of mine. And is currently in progress!
In this project I will list all movie, books, series, games etc. titles on display. The user while loged in, will be able to mark as watched or interested. Furthermore, he will be able, if he watched it, to hand a review about it. He then will be able to access the list of watched, interested, played, read etc. via a easy set lists. The reason for this is, that from my own experience, that after watching, reading etc. its common to forget about it very easy. Since, sometimes we might not come back to it, it is kind of leaving our brain and next time we step into it, we crash our brains, if we watched it or not. That why I am here! I don't know if someone has done this already, but for me it is a start idea for a new project, something to start learning scripts. Yes, I am speaking about html, css and maybe javascript, phyton, sql etc.
